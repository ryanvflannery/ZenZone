ZenZone
ZenZone is a focus-driven accountability app designed to help users actually get into the zone, complete goals, and stay consistent over time.

🚀 Features
✅ Zen Mode – Minimize distractions and enter a deep work state.
✅ Goal Tracking – Set, track, and complete both short-term and long-term goals.
✅ Consistency Metrics – Monitor your progress over time to build lasting habits.

🛠 Tech Stack
Frontend: TypeScript, React Native, Expo, NativeWind CSS
Backend: TypeScript, Node.js, Express
Database: MySQL
